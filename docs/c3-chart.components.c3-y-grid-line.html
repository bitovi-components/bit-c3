<c3-y-grid-line>

  • module
c3-chart.components.c3-y-grid-line

{can.Component}

 
  • source

Chart Y grid line element

<c3-y-grid value="" text="" position="" class="" />

Parameters

  1. value {Number}

    Value where the grid line will start from

  2. text {String}

    Label for the grid line

  3. position {String}

    Position of the grid line

  4. class {String}

    Class name applied to the label

Component Initialization

<c3-chart>
  <c3-y-grid>
    <c3-y-grid-line value="0" text="Break Even" class="label-0" position="middle" />
  </c3-y-grid>
</c3-chart>
