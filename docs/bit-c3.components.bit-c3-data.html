<bit-c3-data>

  • module
bit-c3.components.bit-c3-data

{can.Component}

 
  • source

Chart data element

<bit-c3-data></bit-c3-data>

Parameters

  1. type {String}

    Type of the graph (see http://c3js.org/reference.html#data-type).

  2. names {Object}

    Names of the data elements, keyed by the data Key (see http://c3js.org/reference.html#api-data-names).

  3. url {String}

    Load a CSV or JSON file from a URL (see http://c3js.org/reference.html#data-url).

  4. json {Object}

    Parse a JSON object for data (see http://c3js.org/reference.html#data-json).

  5. columns {Array}

    Load series data from a multidimensional array (see http://c3js.org/reference.html#data-columns).

  6. rows {Array}

    Load row data from a multidimensional array (see http://c3js.org/reference.html#data-rows).

  7. classes {Object}

    Set custom data classes (http://c3js.org/reference.html#data-classes).

  8. categories {Array}

    Set custom X axis categories (see http://c3js.org/reference.html#axis-x-categories).

  9. axes {Object}

    Relate data to an axis (see http://c3js.org/reference.html#api-data-axes).

  10. colors {Object}

    Set color for data (see http://c3js.org/reference.html#data-colors).

  11. types {Object}

    Set types for data series (see http://c3js.org/reference.html#data-types).

  12. unload {String | Array}

    Data ID(s) to be unloaded.

  13. done {function()}

    Function that will be ran after data is loaded.

Component Initialization

<bit-c3>
  <bit-c3-data></bit-c3-data>
</bit-c3>
